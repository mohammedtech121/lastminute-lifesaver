## 🧠 Project Overview & Hackathon Focus
[cite_start]Traditional productivity systems rely on passive reminders that users easily dismiss or ignore[cite: 43]. **The Last-Minute Life Saver** solves this by embedding an autonomous agentic framework driven by **Google AI Studio** and **Gemini 1.5 Flash**. [cite_start]Instead of static push alerts, it parses dynamic textual user deadlines, computes real-time urgency weights, decomposes tasks into immediate structured execution trees, and identifies proactive failure blocks[cite: 45, 48, 55].

### 💎 Key Engineering Features
* [cite_start]**Intelligent Task Prioritization:** Contextually scores incoming inputs into absolute priority grids (High/Medium/Low) based on timeline parameters[cite: 48].
* **Agentic Execution Trees:** Automatically structures chaotic descriptions into clear, sequential HTML task-action plans without human prompting parameters.
* **Proactive Blocker Insights:** Leverages generative AI forecasting to flag real-world friction bottlenecks before they derail user operations.
* **Secure Serverless Core:** Implements a zero-leak proxy architecture via a Node.js serverless back-end handling secure API endpoints through environment variables.

## 🛠️ Technology Stack
* **Core LLM Processing:** Google AI Studio Framework & Gemini 1.5 Flash API
* **Frontend Workflow:** HTML5, Modern CSS3 Core, Vanilla ES6 JavaScript
* **Backend Architecture:** Vercel Node.js Serverless Functions Proxy
